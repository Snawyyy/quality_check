# Quality Check GUI - Build Information

## Executable
The standalone executable `QualityCheckGUI.exe` has been created in the `dist` folder.

## Usage
1. Double-click `QualityCheckGUI.exe` to run the application
2. Use the file dialogs to select:
   - Complot CSV file
   - Layer Excel file
   - Recommendations template Excel file
   - Output location for the results
3. Click "Run Quality Check" to process the files
4. View results in the application interface

## Notes
- The executable is ~115MB and contains all necessary dependencies
- No installation required - it's a standalone application
- Requires Windows OS to run
- For best performance, ensure you have sufficient RAM when processing large files
